> # Nature Powered Computer v2.0  
**Intelligent Hybrid Off-Grid Raspberry Pi System with Real-time AI & Self-Healing**

**School:** CM Shri School, Sector-6, Dwarka, Delhi  
**Hack Club Blueprint** • March 2026

### 🎯 One-Line Description
A completely self-sustaining Raspberry Pi computer that runs 24/7 using only natural energy sources (Solar as main + TEG + Wind + Piezo) with real-time power dashboard, local AI for Mars-like feature detection, and a self-healing demo.

### Why This Project?
Common school projects use only solar. I wanted to build something **smarter and future-ready**:
- Multiple natural energy sources combined intelligently
- Never dies even if one source is weak (smart load-first priority)
- Local AI + self-healing demo (inspired by future Mars missions)
- Zero grid dependency → useful for disaster zones, remote areas, and space

This is not just a charger — it's an **intelligent off-grid brain** for future robots.

### Key Features (v2.0 Upgrades)
- Real-time OLED Power Dashboard (live wattage of every source)
- Local AI (TensorFlow Lite + Pi Camera) → detects Mars-like rocks, ice patches, safe landing zones
- Self-Healing Demo (vibration → piezo recharge + "heal" LED)
- Wireless Monitoring (ESP32 → mobile app)
- Smart Load-First Priority Logic
- Modular Plug-and-Play design
- Mini Mars Arena Demo box

### How It Works (High Level)
1. Solar (main source) + TEG + Wind + Piezo harvest energy
2. MPPT + buck-boost + supercaps + LiFePO₄ battery
3. Raspberry Pi Zero 2W gets stable 5V always (load-first logic)
4. Pi runs local AI and shows live data on OLED
5. Self-healing module shows piezo can recover from "damage"
